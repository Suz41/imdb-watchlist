# IMDb Watchlist & Achievements Tracker

A collection of web utilities for cataloging your movie list and tracking completion of the IMDb Top 500 films, styled with a Criterion Collection dashboard aesthetic.

## Deployed Live Apps

- 📊 **[IMDb Watchlist Manager](https://suz41.github.io/imdb-watchlist/)**
- 🏆 **[IMDb Top 500 Achievements Tracker](https://suz41.github.io/imdb-watchlist/top500.html)**

## Features

- **Watchlist Manager (`index.html`)**: Personal movie logger with local storage persistence, statistics tracker (runtime, genre, counts), and autocomplete movie suggestions.
- **Top 500 Achievements (`top500.html`)**: Gamified checklist parsing the Top 500 movies of the decade, featuring 9 unlockable badges and progress indicators.
- **Criterion Vibe**: Styled using Outfit and Playfair Display typography with glassmorphic cards and gold accents.

## Local Running

Start a local server to view:
```bash
python -m http.server
```
Visit `http://localhost:8000`.
